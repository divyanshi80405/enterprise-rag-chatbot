# Enterprise RAG Chatbot

A Retrieval-Augmented Generation (RAG) application built using Python, LangChain, Gemini, and FAISS for answering questions from uploaded PDF documents.

## Features

* PDF Upload
* Text Extraction using PyPDF
* Document Chunking
* Embedding Generation
* FAISS Vector Database
* Semantic Search
* Gemini-powered Question Answering

## Tech Stack

* Python
* LangChain
* Google Gemini
* FAISS
* Sentence Transformers
* FastAPI

## Architecture

PDF → Text Extraction → Chunking → Embeddings → FAISS → Retrieval → Gemini → Answer

## Example Questions

* What role is this resume best suited for?
* What are the candidate's strongest skills?
* Summarize this document.
* What projects are mentioned?
